## 4. Списки
### Маркированный список:
* Первый пункт
+ Второй пункт
- Третий пункт

### Нумерованный список:
1. Первый элемент
2. Второй элемент
3. Третий элемент

### Вложенный список:
* Основной пункт 1
  + Вложенный пункт 1.1
  * Вложенный пункт 1.2
- Основной пункт 2
  1. Вложенный нумерованный 2.1
  2. Вложенный нумерованный 2.2
